#  vim-config

## Installation

```
git clone https://github.com/gradel/vim-config.git ~/.vim
```

```
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

Link to .vimrc

```
cd $HOME && ln -s .vim/.vimrc && ln -s .vim/.gvimrc
```

Launch vim and

```
:BundleInstall
```

watch Vundle do all the work :).

You're done.
